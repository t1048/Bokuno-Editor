# Bokuno-Editor

極限まで「軽さ」と「速さ」を追求したテキストエディタ＆高速検索ツールです。
機能を絞り込むことで、圧倒的な起動速度と低メモリ消費を実現しています。

## 特徴

- **圧倒的な軽量・高速動作**: Tauri (Rust) を基盤とし、Chromiumを同梱しないことで、瞬時の起動と軽快な動作を実現。
- **Live Tail (ログ監視)**: ファイルの末尾をリアルタイムに追従して表示。サーバーログやアプリケーションログの監視に最適。
- **ファイルプレビュー**: CSVファイルの表形式プレビューおよびMarkdownファイルのレンダリングプレビューに対応。
- **大ファイルのストリーム読み込み**: 大きなファイルもメモリを圧迫せずに開けるストリーミング読み込みに対応（5MB 超で自動的にストリーム処理）。
- **ワークスペースとしてフォルダを開く**: プロジェクト単位でフォルダをワークスペースとして開き、ファイルツリー・検索を管理。最後に開いたワークスペースは次回起動時に復元されます。
- **設定の永続化**: テーマ、フォントサイズ、エンコーディング、改行コードなどの設定は再起動後も保持されます。
- **最近使ったファイル**: 直近で開いたファイルを素早く再オープンできます。
- **高速全文検索 (Grep)**: Rust製の検索エンジンにより、大規模なプロジェクト内も一瞬で全文検索。正規表現にも対応。
- **シンタックスハイライト**: CodeMirror 6を採用し、Markdown, Python, C++, Rust, JavaScript, HTML, CSS, YAML, Go などに対応。
- **OS統合 (Windows)**: エクスプローラーの右クリックメニューから、ファイルを手軽に開いたり、フォルダ内を即座にGrep検索可能。
- **モダンなUI/UX**: シンプルかつ直感的なインターフェース。ダークモード完備。

## 技術スタック

- **コア (バックエンド)**: [Tauri 2](https://tauri.app/) (Rust) - メモリセーフで高速。
- **UI (フロントエンド)**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- **エディタエンジン**: [CodeMirror 6](https://codemirror.net/) - モジュール式で超軽量。
- **スタイリング**: [Tailwind CSS 4](https://tailwindcss.com/) - 最適化されたユーティリティファーストCSS。

## 使い方

### インストール

```bash
# 依存関係のインストール
npm install

# 開発モードで起動
npm run tauri dev

# 本番用バイナリのビルド
npm run tauri build
```

### ショートカット

| ショートカット | 機能 |
|--------------|------|
| Ctrl + N | 新規ウィンドウ |
| Ctrl + O | ファイルを開く |
| Ctrl + S | 保存 |
| Ctrl + Shift + S | 名前を付けて保存 |
| Ctrl + F | 現在のファイル内検索 |
| Ctrl + H | 置換 |
| Ctrl + Shift + F | ワークスペース Grep 検索 |
| Ctrl + ホイール | フォントサイズ変更 |
| F2 / Shift + F2 | 次/前の検索結果 |

### 便利機能

- フォルダをワークスペースとして開く: ツールバーからフォルダを開き、サイドバーのファイルツリーで管理できます。
- 外部ファイル変更の検知: 編集中のファイルが他のアプリで更新された場合、再読み込みを確認します（未保存の変更がある場合は保護されます）。
